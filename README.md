# smart-minds/remote-server

remote server for facebook user management, phone numbers and url for access to the home server


### Installing

First install node_modules

```
npm install
```

Second start your mongodb server

``` bash
mongod
```

## Running the tests

To run the test, just type 
``` 
npm run start
```

## Deployment

To deploy on heroku after commit all
``` 
git push heroku master
```
